This is the constructor you would call if you wanted to create an AssemblyScratch object explicitly and loop over all the cells and assemble the local matrices and vectors by hand. This would be done sequentially.

template<int dim>
Assembly::AssemblyScratch< dim >::AssemblyScratch ( const AssemblyScratch< dim > &  scratch)

This is the copy constructor that will be called in the WorkStream call that will build and distribute the local matrices and vectors in parallel using Thread Building Blocks.
