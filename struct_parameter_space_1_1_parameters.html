Detailed Description

Struct which holds the parameters used for simulations.

These are the parameters which will be used by DriftDiffusionPoisson class to do simulations. They will be read read in using the ParameterReader class to read them from input_file.prm and then use the ParameterHandler to set the variables and scale them if necessary.
