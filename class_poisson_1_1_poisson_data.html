Detailed Description

template<int dim>
class Poisson::PoissonData< dim >

Data structures, functions/paramaters, and solver for Poisson equation.

This object holds the matrix, vectors, constrains and solver for the mixed finite element approximation to Poisson's equation. See MixedPoisson::MixedFEM for more information on the mixed method.

Member Data Documentation

template<int dim>
SparseMatrix<double> Poisson::PoissonData< dim >::system_matrix

The matrix will be of the form,

\[ \left[ \begin{matrix} A & B \\ B^{T} & 0 \end{matrix} \right] \]

See MixedPoisson::MixedFEM for more information on the mixed method.

template<int dim>

This will be used to store the right hand side of the mixed method,

\[ = \ \frac{1}{\lambda^{2}} \ \left( v,\ \rho_{n}^{e} \ - \ \rho_{p}^{e} \ -\ \left( \rho_{n} \ - \ \rho_{p} \right) \right) \]

See MixedPoisson::MixedFEM for more information on the mixed method.
